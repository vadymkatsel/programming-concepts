# Основи програмування (Introduction to Programming)

Цей репозиторій містить вихідні файли та матеріали для сайту курсу "Основи програмування". Сайт побудований як статична книга за допомогою [Quarto](https://quarto.org/) і має кастомне CSS-оформлення та окрему uгголовну сторінку

🔗 **[Посилання на сайт курсу](https://vadymkatsel.github.io/programming-concepts/)**

## 🛠 Стек Технологій

* **Quarto:** для генерації статичного сайту з Markdown-файлів.
* **HTML/CSS:** для кастомної стилізації та лендінгу.
* **JavaScript** для підлаштування стилізації та окремих налаштувань вебсайту
* **GitHub Pages:** для хостингу.

## 📂 Структура проєкту

* `_quarto.yml` — головний конфігураційний файл проєкту.
* `index.html` `index.css` — головна сторінка та стилі.
* `styles.css` — стилі, використані для сторінок курсів
* `modules/` — розділи курсу, лекції та практичні завдання.

## 🚀 Як розгорнути локально

Якщо ви хочете зібрати сайт на своєму комп'ютері:

1. Встановіть [Quarto](https://quarto.org/docs/get-started/).
2. Клонуйте цей репозиторій:
   ```bash
   git clone https://github.com/vadymkatsel/programming-concepts

Перейдіть у папку проєкту та запустіть прев'ю:

```bash
quarto preview
```

Щоб згенерувати фінальні статичні файли (зазвичай у папку _site або docs), виконайте:

```bash
quarto render
```
